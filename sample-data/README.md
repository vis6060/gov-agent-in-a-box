# Synthetic Sample Data

All content in this folder is **synthetic** and for demo/testing only.
- Emails use the reserved `example.com` domain.
- Phone numbers use the 555 range.
- SSNs are non-assignable patterns (e.g., 000-12-3456).

Use these seeds to demo:
- Submit tasks that trigger PII redaction (email).
- Enqueue HITL items and exercise review/rollback.
- Show RAG "top docs" for each region.

No production or real personal data is included here.
